# UrbanSense-Citizen-Driven-Micro-Climate-Air-Quality-Monitoring-Platform
UrbanSense is a citizen-driven air quality monitoring platform using low-cost IoT sensors and machine learning. It provides real-time pollution data, interactive dashboards, and personalized health alerts, empowering communities and supporting smarter, healthier, and sustainable cities.
